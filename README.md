## 🚀 Getting Started

### Requirements

- Flutter SDK (>= 3.0.0)
- Firebase account
- Android Studio or Xcode

### Setup Steps

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/homeease.git
   cd homeease
   ```

2. Install packages:
   ```bash
   flutter pub get
   ```

3. Set up Firebase:
   - Add `google-services.json` (Android)
   - Add `GoogleService-Info.plist` (iOS)
   - Enable Firebase Auth & Firestore

4. Run the app:
   ```bash
   flutter run
   ```
